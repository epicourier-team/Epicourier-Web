## ğŸ” Troubleshooting

1. **Gemini API Key Error**

   - Verify your `.env` file contains a valid `GEMINI_API_KEY`
   - Check that the API key has the necessary permissions

2. **JSON Parsing Error**

   - The Gemini model's response may not be valid JSON
   - When errors occur, the raw response is saved to the cache file and a warning is displayed in the logs

3. **Prompt Files Missing**
   - You need to create the `prompts/` directory and required prompt files

## ğŸ“ Notes

- Each recipe processing may take several seconds depending on the Gemini model's response time
- Retries up to 3 times on API errors
- All output is saved in UTF-8 encoding
